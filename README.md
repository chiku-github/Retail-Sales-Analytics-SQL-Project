📌 Project Overview

This project performs Exploratory Data Analysis (EDA) on retail sales data using SQL Server.

It focuses on analyzing customer behavior, product performance, and sales trends to generate business insights.

---

🎯 Business Problem (BP)

Retail businesses often struggle to:

- Identify top-performing products
- Understand customer purchasing behavior
- Track sales growth over time
- Analyze category-wise performance
- Improve business decision-making

---

📊 Key KPIs

- Total Sales
- Total Orders
- Total Customers
- Total Quantity Sold
- Average Price
- Product Performance
- Customer Segmentation

📈 Key Insights

---

🔍 Analysis Performed

1️⃣ Exploratory Data Analysis (EDA)

- Data exploration using SELECT queries
- Checking NULL values & data types
- Understanding table structures

2️⃣ Dimension & Date Analysis

- Unique customers, categories, products
- Date range & customer age analysis

3️⃣ KPI Analysis

- Sales, orders, customers, revenue metrics
- Category & gender-based performance

4️⃣ Ranking Analysis

- Top & bottom performing products
- Top customers by revenue

5️⃣ Time-Based Analysis

- Monthly & yearly sales trends
- Growth analysis

6️⃣ Advanced Analytics

- Running totals (cumulative analysis)
- Segmentation (price-based grouping)
- Year-over-Year (YoY) performance

- 📌 Product Report → Product performance & revenue contribution
